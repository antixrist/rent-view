О проекте SocRent SocRent :: Объявления аренды квартир из социальных сетей

SocRent - это автоматический сервис, который собирает объявления из популярных социальных сетей, сортирует их и показывает в удобном формате с возможностью поиска.


Объявления берутся из сообществ, пабликов, групп и стен пользователей.

Для сортировки используются машинные алгоритмы, поэтому иногда объявления могут быть классифицированны неверно.


На данном этапе сервис работает только для города Санкт-Петербург.
